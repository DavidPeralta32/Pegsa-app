# PEGSA Construcciones

Landing page desarrollada con Vue.js, adaptada y personalizada para Pegsa Construcciones.

## 🛠️ Tecnologías
- Vue.js
- Vite
- Tailwind CSS
- shadcn-vue

## ✨ Descripción
Este proyecto es una landing page moderna y responsiva.  
Fue modificada en estructura, estilos y contenido para ajustarse a un nuevo propósito y branding.
